Box model merupakan kerangka dasar dalam CSS untuk menggambarkan cara elemen-elemen HTML ditampilkan dalam halaman web. Setiap elemen dipresentasikan sebagai kotak yang terdiri dari empat bagian utama: content (isi elemen), padding (jarak antara isi dan border), border (batas luar elemen), dan margin (jarak antara elemen dengan elemen lain di sekitarnya). Pemahaman box model memungkinkan pengembang mengatur posisi dan jarak elemen secara presisi, sehingga tampilan halaman menjadi terstruktur dan harmonis. Misalnya, dengan mengatur padding dan margin, kita dapat menciptakan ruang antar elemen untuk meningkatkan kenyamanan visual.
