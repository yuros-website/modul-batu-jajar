CSS dirancang untuk memberi instruksi kepada browser mengenai bagaimana elemen-elemen HTML harus ditampilkan kepada pengguna. Melalui CSS, seorang pengembang dapat menentukan gaya visual seperti warna teks, jenis dan ukuran font, jarak antar elemen, posisi objek dalam layout, dan sebagainya. Sebagai bahasa stylesheet, CSS tidak berfungsi untuk mengatur logika atau alur program, melainkan fokus pada desain visual, menjadikannya pelengkap dari HTML yang berfungsi sebagai struktur konten.
