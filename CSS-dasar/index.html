Pengantar dan Latar Belakang CSS Dasar

  • Dalam pengembangan web modern, aspek visual memegang peran penting dalam meningkatkan pengalaman pengguna.
  • CSS (Cascading Style Sheets) hadir sebagai bahasa standar yang dirancang untuk memisahkan struktur dokumen HTML dari aspek presentasinya.
  • Penerapan CSS memungkinkan kontrol yang efisien dan konsisten terhadap tampilan antarmuka web, menjadikannya komponen esensial dalam web design.

Definisi dan Fungsi CSS

  • CSS adalah bahasa stylesheet yang digunakan untuk mengatur gaya visual elemen HTML.
  • Fungsinya meliputi pengaturan warna, tipografi, layout, serta adaptasi tampilan berdasarkan perangkat.
  • Dengan CSS, developer dapat menciptakan halaman web yang lebih terstruktur, estetis, dan responsif terhadap berbagai ukuran layar.

Metode Penulisan CSS

  • Inline CSS: langsung pada elemen HTML; digunakan untuk styling cepat namun tidak disarankan untuk proyek besar.
  • Internal CSS: ditulis di dalam tag style dalam dokumen HTML.
  • External CSS: ditulis dalam file terpisah dengan ekstensi `.css` dan dihubungkan ke HTML melalui tag ``.

Struktur dan Selektor CSS

  • Struktur sintaks CSS terdiri dari *selector*, *property*, dan *value*.
  • Selector menentukan elemen mana yang akan diberi gaya, sementara properti dan nilainya menentukan gaya seperti warna, ukuran, dan margin.
  • CSS mengenal berbagai jenis selector seperti element, class, ID, dan kombinasi (descendant, group).

Penerapan dan Konsep Lanjutan

  • Konsep box model menjelaskan bagaimana elemen HTML dipresentasikan sebagai kotak yang terdiri dari margin, border, padding, dan content.
  • CSS juga memungkinkan desain yang responsif melalui media queries, sehingga tampilan halaman dapat menyesuaikan dengan resolusi perangkat.
  • Pemahaman terhadap prinsip dasar ini menjadi landasan untuk pengembangan CSS yang lebih kompleks seperti Flexbox, Grid, dan animasi.
