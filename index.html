Installasi git pada lingkungan windows

Oleh: Muhammad Farhan

Apa itu git

Git adalah sistem kontrol versi (version control system) yang digunakan untuk melacak perubahan dalam file atau kode selama proses pengembangan perangkat lunak. Git memungkinkan banyak orang bekerja dalam satu proyek secara bersamaan tanpa menimpa pekerjaan satu sama lain.

Persiapan installasi

Sebelum menginstal Git di Windows, pastikan perangkat telah memenuhi persyaratan dasar. Instalasi Git cukup ringan dan tidak membutuhkan spesifikasi tinggi, namun koneksi internet tetap dibutuhkan untuk mengunduh file instalasi.

Unduh git

  • Kunjungi situs: [https://git-scm.com](https://git-scm.com)
  • Pilih versi Git untuk Windows
  • Klik “Download”
    •

    Proses installasi git

    • Jalankan file installer (.exe)
    • Langkah-langkah wizard: - Pemilihan komponen - Path Environment - Editor default (misal: Vim atau Notepad++) - Pilihan terminal (Git Bash vs CMD)

    Konfigurasi awal git

    • git config --global user.name "Nama Anda"
    • git config --global user.email "email@contoh.com"

    Setelah instalasi selesai, langkah selanjutnya adalah melakukan konfigurasi awal Git. Ini bertujuan agar setiap perubahan yang kamu buat dapat dikenali identitasnya oleh Git, terutama saat bekerja dalam tim atau menggunakan layanan seperti GitHub.

    Verifikasi installasi

    • Cek versi Git
    • Tes Git Bash & CMD
    • Konfirmasi konfigurasi

    Penutup & Saran Lanjutan

    Setelah proses instalasi dan konfigurasi selesai, Git sudah siap digunakan untuk mengelola versi proyek Anda. Langkah berikutnya adalah mempelajari perintah dasar Git serta memahami cara berkolaborasi menggunakan platform seperti GitHub.

    kesimpulan

    Git adalah alat penting untuk mengelola versi proyek secara efisien, terutama dalam pengembangan perangkat lunak. Proses instalasinya di Windows cukup mudah, dimulai dari mengunduh installer, menjalankan setup, hingga melakukan konfigurasi awal. Setelah terpasang dan terverifikasi, Git siap digunakan untuk melacak perubahan, bekerja dalam tim, dan berkolaborasi melalui platform seperti GitHub. Untuk hasil optimal, pengguna dianjurkan mempelajari perintah dasar Git sebagai langkah lanjutan.
