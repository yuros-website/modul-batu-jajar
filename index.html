Apa Itu Markdown

Markdown adalah bahasa markup ringan (lightweight markup language) yang digunakan untuk menulis teks dengan format sederhana yang bisa dengan mudah diubah menjadi HTML (HyperText Markup Language).

Kelebihan Markdown

  • Sintaks sederhana dan mudah diingat
  • Mudah dikonversi ke HTML, PDF, dan lainnya
  • Banyak digunakan di GitHub, Reddit, dan platform dokumentasi
  • Ringan dan kompatibel lintas platform
    •

    Cara Menggunakan Markdown

  • Menggunakan editor teks biasa atau editor khusus Markdown
  • Pratinjau bisa dilihat secara langsung di banyak IDE dan platform seperti VS Code, GitHub, Notion, dll.
    •

    Sintaks Dasar

  • **Teks** atau __Teks__
  • *Italic* atau _Italic_
    •

    Markdown dirancang untuk membuat penulisan teks lebih sederhana dan bersih, terutama saat membuat dokumen atau konten web.

    Link dan Gambar

  • [Kunjungi GitHub](https://github.com)

    Markdown menyediakan cara yang sangat sederhana untuk menambahkan tautan (link) dan gambar ke dalam dokumen, tanpa harus menulis kode HTML yang panjang.

    Tools & Editor Markdown

    • Online: Dillinger, StackEdit
    • Offline: Typora, Obsidian, VS Code
    • Converter: pandoc untuk ke PDF/HTML

    Tips Menulis Markdown yang Efektif

    • Gunakan heading untuk struktur
    • Gunakan list untuk poin-poin
    • Manfaatkan preview editor
    • Simpan dalam `.md`

    kesimpulan

    Markdown adalah bahasa markup ringan yang mudah digunakan untuk membuat dokumen terstruktur. Dengan sintaks sederhana, Markdown memudahkan penulisan teks, dokumentasi, dan konten web tanpa perlu HTML. Cocok digunakan di berbagai platform seperti GitHub, Notion, dan VS Code karena fleksibel dan efisien.
