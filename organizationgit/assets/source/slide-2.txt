GitHub adalah sebuah platform berbasis web yang digunakan untuk menyimpan, mengelola, dan berbagi kode program secara online. Platform ini dibangun di atas sistem kontrol versi yang bernama Git, sebuah alat yang memungkinkan pengguna untuk melacak perubahan dalam file dari waktu ke waktu, bekerja secara kolaboratif tanpa risiko kehilangan data, dan kembali ke versi sebelumnya bila diperlukan. GitHub sangat populer di kalangan programmer, pengembang perangkat lunak, peneliti, bahkan penulis dokumentasi teknis karena memudahkan kolaborasi dan pengelolaan proyek secara sistematis.

Saat seseorang membuat proyek baru di GitHub, proyek tersebut akan disimpan dalam sebuah wadah yang disebut repository (atau repo). Repository ini bisa diatur agar bersifat publik—terbuka untuk siapa pun di dunia—atau privat—hanya bisa diakses oleh pengguna tertentu. Dalam repository, setiap perubahan yang dilakukan pada file harus dicatat melalui proses yang disebut commit, yang berfungsi seperti “catatan versi” dari setiap perbaikan, penambahan, atau penghapusan. Jika seorang pengguna ingin mencoba ide baru atau mengembangkan fitur tertentu tanpa mengganggu versi utama proyek, ia dapat membuat branch, yaitu salinan cabang dari proyek utama. Setelah perubahan di branch selesai dan ingin digabungkan kembali ke versi utama, pengguna akan membuat pull request—permintaan untuk menggabungkan perubahan tersebut. Fitur ini menjadi kunci utama dalam kolaborasi tim di GitHub.

GitHub telah menjadi bagian penting dalam dunia teknologi karena memudahkan kolaborasi lintas negara, organisasi, dan bahkan antar individu. Banyak perusahaan teknologi besar seperti Google, Microsoft, dan Meta juga menggunakan GitHub untuk mengelola proyek open-source mereka. Bagi pelajar atau profesional, GitHub bisa menjadi portofolio digital yang menunjukkan keterampilan dan kontribusi mereka dalam dunia pemrograman.
