Repository (atau sering disingkat repo) di GitHub adalah tempat penyimpanan utama untuk seluruh berkas dan riwayat versi suatu proyek, baik itu kode program, dokumentasi, maupun file pendukung lainnya. Secara sederhana, repository bisa diibaratkan seperti folder proyek digital yang tersimpan di cloud, dan bisa diakses kapan saja oleh individu atau tim. Setiap kali kita membuat perubahan dalam proyek, perubahan tersebut bisa dicatat dan dilacak melalui commit, sehingga seluruh histori perkembangan proyek terdokumentasi dengan baik. Dengan kata lain, repository bukan hanya tempat menyimpan kode, tetapi juga sistem pelacakan perubahan, alat kolaborasi, dan ruang dokumentasi dalam satu kesatuan.

Di GitHub, setiap repository memiliki halaman khusus yang menampilkan struktur file, deskripsi proyek, kontribusi anggota, dan alat-alat untuk kolaborasi seperti issue tracking dan pull request. Pengguna dapat membuat repository untuk proyek apa pun, mulai dari skripsi, website pribadi, aplikasi mobile, hingga proyek skala besar seperti framework atau sistem backend. Baik untuk pengembangan individu maupun kerja tim, repository menjadi fondasi utama dalam alur kerja Git berbasis GitHub.
