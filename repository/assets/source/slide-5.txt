Membuat repository di GitHub adalah langkah awal untuk memulai proyek baru secara online dan menyimpannya di cloud. Proses ini sangat sederhana dan bisa dilakukan langsung dari dashboard akun GitHub. Pertama, setelah login, klik tombol “+” di pojok kanan atas dan pilih “New repository”. Selanjutnya, Anda akan diarahkan ke halaman pembuatan repositori. Di sana, Anda perlu mengisi beberapa informasi penting, seperti nama repository, yang akan menjadi identitas proyek Anda di URL (misalnya https://github.com/username/nama-repo). Disarankan untuk menggunakan nama yang deskriptif dan singkat agar mudah diingat dan dikenali.

Anda juga bisa menambahkan deskripsi singkat tentang tujuan proyek pada kolom deskripsi (opsional, tetapi sangat direkomendasikan untuk mempermudah pemahaman pengguna lain). Setelah itu, pilih jenis repositori: Public jika ingin bisa diakses siapa pun, atau Private jika hanya ingin dibagikan ke orang tertentu.

GitHub juga memberikan opsi untuk menginisialisasi repository dengan beberapa file dasar. Anda bisa mencentang pilihan untuk menambahkan file README.md, agar repositori langsung memiliki dokumentasi awal; file .gitignore, untuk mengecualikan file tertentu dari pelacakan Git; dan lisensi (LICENSE) jika Anda ingin menentukan izin penggunaan kode sejak awal. Setelah semua pilihan disesuaikan, klik tombol “Create repository”.

Setelah repositori berhasil dibuat, Anda akan diarahkan ke halaman utama repositori tersebut, dan dari sana Anda bisa mulai mengunggah file, mengkloningnya ke komputer lokal, atau mengundang kolaborator. Langkah ini menjadi pondasi awal sebelum Anda mulai menambahkan konten atau bekerja secara tim.
