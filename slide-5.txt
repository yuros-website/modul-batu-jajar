Setelah file installer Git berhasil diunduh, tahap berikutnya adalah proses instalasi yang dilakukan melalui wizard instalasi. Pengguna memulai dengan menjalankan file .exe yang telah diunduh. Jendela wizard akan terbuka dan menampilkan serangkaian langkah pengaturan instalasi. Langkah pertama adalah menyetujui perjanjian lisensi, yang wajib diterima untuk melanjutkan instalasi. Setelah itu, pengguna dapat memilih lokasi folder tujuan instalasi, biasanya secara default akan diarahkan ke direktori C:\Program Files\Git, namun lokasi ini bisa diubah sesuai kebutuhan.

Pada tahap berikutnya, wizard akan meminta pengguna memilih komponen tambahan yang ingin diinstal, seperti integrasi dengan Windows Explorer (menu klik kanan), ikon desktop shortcut, dan pilihan editor teks default yang akan digunakan Gitâ€”umumnya disarankan memilih Notepad++ atau Vim, tergantung preferensi. Selanjutnya, pengguna menentukan pengaturan PATH environment, yaitu apakah Git akan tersedia dari Git Bash saja atau juga dapat diakses melalui Command Prompt Windows (CMD). Supaya lebih praktis, banyak pengguna memilih opsi agar Git dikenali di semua terminal.

Wizard juga menyediakan pengaturan tambahan terkait line ending conversion, yang penting untuk kompatibilitas file teks antar sistem operasi (Windows, macOS, Linux), serta pilihan emulator terminal, misalnya menggunakan Git Bash atau terminal bawaan Windows. Setelah semua pengaturan dikonfirmasi, pengguna cukup klik Install, dan proses penyalinan file ke sistem akan berjalan secara otomatis hingga selesai. Proses ini biasanya hanya memerlukan waktu beberapa menit. Setelah instalasi selesai, akan muncul opsi untuk langsung menjalankan Git Bash atau membaca dokumen rilis, yang menandakan Git siap digunakan.
